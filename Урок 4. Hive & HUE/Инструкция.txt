По уроку проходим пока инструкцию HDFS

	!!! Как отправить файл в HDFS от ПК, и получить файл из HDFS к ноду с помощью команды -GET.

1. Нам надо отправить файл от лаптопа или компа к серваку(Менеджеру или калстеру или ноду)
с помощью команды показано нж:

> $scp -i key.txt file.txt student10_16@novalocal
потом наш файл должен оказаться в кластере, но не в HDFS
Если что можно посмотреть инструкцию 2 го урока, там достоверно написано.

2. Теперь с кластера надо отпрваить в HDFS тут все проще
с помощью команды показано нж:

> hdfs dfs -put file.txt
С этой командой мы отправляем файл в HDFS. Оказывается HDFS это как облако что ли, сеть где файлы
будет в общей доступе.

3. Продожение 2 го этапа. Как сказано выше, файлы доступны любому обще соединенному кластерут если ону выложены
в HDFS, то есть мы можем скачать файл с другого кластера из HDFS.
с помощью команды показано нж:

> hdfs dfs -get file.txt
заметьте я пишу из другого кластера или нода. Таким образом я скачаю файл из HDFS в кластер.



 - + - + - И так оснавная задача создать базу данных и залить готовую таблицу в базу  - + - +  -

1. Готовим таблицу, то есть скачаем или создаем на ваш выбор.

2. Отправляем в кластер через 
> scp -i id_rsa_gb.txt FILE.format student10_16@novalocal:.

3. Отправляем в hdfs через
> hdfs dfs -put

4. Заходим в hive через команду < hive > и создаем Базу через
> create database databaseName;

5. Заливаем таблицу в базу, но есть некотрые моменты. Перед тем как залить нужно писать команду
> create external table stud16.db (id int, cntr string, ecr string) 
< Row format delimited fields terminated by ',' 
< location '/user/student10_16/convr';

То есть создадим таблицу называние db, stud16 это называние базы. Потом укажем колонок.
Затем пишем что нужно перейти на следующую колонку когда находиш ","
Потом укажем путь где нужно сохранить файл таблицы. Да Да!!! Таким образом создадим папку для таблицы
После чего туда переместим CSV файл или который мы готовили через команду
> hdfs dfs -mv filename.format dirname

6. Проверяем, пишем запрос в HIVE 
> select * from tablename;
