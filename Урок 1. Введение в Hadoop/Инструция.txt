смог зайти через терминал git bash где есть ssh
Надо было указать формат файла id_rsa_gb.txt дебил

> $ ssh -i id_rsa_gb.txt student10_16@89.208.221.132


В папке C:\Windows\System32\drivers\etc
есть файл hosts
Нужно открыть через блокнот.
Этот файл позволяет подключиться с именами хоста 

Например нужно написать: 89.208.221.132 novalocal
Оставляю это как пример можеш зайти и проверить.
после этого можно написать
 
> $ ssh -i id_rsa_gb.txt student10_16@novalocal

При подключений создается файл known_hosts без формата. В нем хранится данные о подключений.
Если имените IP адрес и ключ в текущем файле а не создадите другой, тогда система даст ошибку о том
что якобы вас кто-то пытается взломать или изменился ключ или адрес надо исправить.
Тут надо удалить файл known_hosts, таким образом вы удалите все данные о подключений или логи (надо дополнить инфо).
